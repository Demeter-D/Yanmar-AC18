# Yanmar AC-18 — Field Service Manual

A self-contained, mobile-friendly field-service manual for the Yanmar AC-18
compact tracked utility tractor (grey-import to the UK). Built around the
3-cylinder Yanmar 3TNE74 diesel and twin-pump hydrostatic drive.

Covers the full service schedule, step-by-step procedures for filters,
HST service, track maintenance, adjustments, and inspections, plus UK
parts cross-references, torque values, and troubleshooting.

## Files

- `index.html` — single self-contained page. Open this in a browser.
- `apple-touch-icon.png` — 180×180 icon for iOS "Add to Home Screen".
- `og-image.png` — 1200×630 social preview (Facebook / Twitter / iMessage).

Upload all three to the repo root. The favicon for desktop tabs is inlined
inside `index.html` (no separate file needed).

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `Yanmar-AC18`).
2. Upload `index.html` to the repository root.
3. Repo → **Settings → Pages**
   - **Source:** *Deploy from a branch*
   - **Branch:** `main` / `/ (root)`
   - Save.
4. Wait about a minute, then open
   `https://<your-username>.github.io/<repo-name>/` on any device.

## Open on iPhone

- Tap the share icon and **Add to Home Screen** for an app-style launch.
- The page works offline once Safari has cached it.
- The hamburger menu (top right) opens the section drawer.

## Notes

- The AC-18 was a Japanese-domestic model and was never officially sold
  in the UK. Yanmar refined the AC-series and the 3TNE74 engine across
  the production run, so part numbers can vary. ALWAYS verify against
  the engine and chassis plates before ordering.
- The HST (hydrostatic transmission) is the most expensive thing on the
  machine — use only fluids that meet JD J20D / Yanmar HST specification.
- Service log entries persist in your browser's local storage; clearing
  Safari data will erase them.
